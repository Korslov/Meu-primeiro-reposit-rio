Principais Tags do HTML com Exemplos

Aqui você encontrará explicações e exemplos práticos para as principais tags do HTML, organizadas por categoria.

1. Estruturais

Essas tags definem a estrutura principal de um documento HTML:

  • <!DOCTYPE>: Declara o tipo de documento como HTML5. 
    <!DOCTYPE html>
  • <html>: A tag raiz que envolve todo o documento HTML. 
    <html lang="pt-BR">...</html>
  • <head>: Contém metadados da página. 
    
<head>
  <meta charset="UTF-8">
  <title>Meu Site</title>
</head>
  • <body>: Abriga o conteúdo visível da página. 
    
<body>
  <h1>Bem-vindo ao meu site!</h1>
</body>

2. Cabeçalhos e Títulos

Usados para destacar hierarquias e organizar o conteúdo:

  • <h1> a <h6>: Representam níveis de título. 
    
<h1>Título Principal</h1>
<h2>Subtítulo Importante</h2>
<h3>Subseção</h3>

3. Texto e Formatação

Essas tags são usadas para adicionar textos e aplicar estilos básicos:

  • <p>: Define um parágrafo. 
    <p>Este é um parágrafo de exemplo.</p>
  • <strong>: Destaca texto com ênfase forte (negrito). 
    <strong>Texto destacado em negrito.</strong>
  • <em>: Aplica ênfase ao texto (itálico). 
    <em>Texto em itálico para ênfase.</em>
  • <br>: Insere uma quebra de linha. 
    Texto na linha 1.<br>Texto na linha 2.

4. Listas

Permitem organizar informações em formato de lista:

  • <ul>: Lista não ordenada. 
    
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
  • <ol>: Lista ordenada. 
    
<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
</ol>

5. Imagens e Mídia

Tags utilizadas para exibir conteúdo visual e de mídia:

  • <img>: Exibe imagens. 
    <img src="imagem.jpg" alt="Descrição da imagem">
  • <video>: Insere vídeos. 
    
<video src="video.mp4" controls>Seu navegador não suporta vídeo.</video>

6. Links e Navegação

Essas tags criam links para outras páginas ou seções:

  • <a>: Define um link. 
    <a href="https://www.exemplo.com">Visite o site</a>
  • <nav>: Representa uma área de navegação. 
    
<nav>
  <a href="#secao1">Seção 1</a>
  <a href="#secao2">Seção 2</a>
</nav>

7. Estruturas de Tabela

Organizam dados tabulares:

  • <table>: Cria uma tabela. 
    
<table>
  <tr>
    <th>Coluna 1</th>
    <th>Coluna 2</th>
  </tr>
  <tr>
    <td>Dado 1</td>
    <td>Dado 2</td>
  </tr>
</table>

8. Formulários

Permitem a coleta de dados do usuário:

  • <form>: Define um formulário. 
    
<form action="/enviar" method="POST">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome">
  <button type="submit">Enviar</button>
</form>
